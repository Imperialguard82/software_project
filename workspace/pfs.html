Title: FYP Allocation Tool

Proposal:

This project calls for the development of a web-based portal wherein NCI academics can upload project ideas for viewing by students.
The students would then be able to select a set of projects (in order of preference) which they would like to undertake.
There would then be an automated allocation process which would allocate the projects. This allocation should endeavor to make as many people happy as possible.

Abstract details: User Modelling, AI scheduling, Similarity Indexes
